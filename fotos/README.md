# Fotos-Ordner

Hier legst du vor dem ersten Trockenlauf deine Bücherregal-Fotos ab. Empfehlung:

- **Anzahl:** 15 bis 25 Fotos.
- **Inhalt:** Nahaufnahmen der Buchrücken aus deinem eigenen Regal. Jedes Foto kann mehrere Bücher enthalten — das zeigt, dass die App auch Gruppen korrekt aufschlüsselt.
- **Mischung:** bewusst einige nicht perfekte Aufnahmen mitnehmen (leichte Unschärfe, verwinkelte Regalreihen, teilweise verdeckte Rücken). Das unterstreicht im Vortrag, dass die App mit realistischer Eingabequalität umgeht — und zeigt ehrlich, wo die Grenzen liegen.
- **Format:** JPEG, PNG oder WebP. Auflösung egal, die Extraktion skaliert auf maximal 2048 Pixel.
- **Dateinamen:** belanglos, werden nur zur Gruppierung verwendet. Das Skript akzeptiert alles außer Leerzeichen-Freaks (Umlaute gehen).

Dieser Ordner wird vom Demo-Skript `extract.py` als Eingabequelle verwendet.
